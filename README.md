# Travl Plannr

A modern travel planning application that helps users organize and manage their trips efficiently.

## Description

Travl Plannr is a web application designed to simplify travel planning. Users can create trips, build detailed itineraries, track expenses, and collaborate with travel companionsâ€”all in one place.

## Tech Stack

### Frontend
- **Next.js 14+** (React framework with App Router)
- **TypeScript** (for type safety)
- **Tailwind CSS** (for styling)

### Backend
- **Next.js API Routes** (serverless functions)

### Database
- **PostgreSQL** (via Supabase or Railway for hosting)
- **Prisma** (ORM for database management)

### Authentication
- **NextAuth.js** (authentication solution)

### Additional Tools
- **Git** (version control)
- **Vercel** or **Railway** (deployment)

## MVP Features

### MVP 1: User Authentication & Trip Creation
- User registration and login
- Create, view, edit, and delete trips
- Basic trip information (name, destination, dates, description)
- User dashboard showing all trips

### MVP 2: Itinerary Planning
- Add activities/events to trip days
- Drag-and-drop or manual ordering of activities
- Activity details (name, time, location, notes)
- Calendar view of trip timeline

### MVP 3: Budget Tracking
- Add expenses to trips
- Categorize expenses (transportation, accommodation, food, activities, etc.)
- View total budget and remaining budget
- Basic expense reports/summaries

## Future Features (Post-MVP)
- Collaboration (share trips with friends/family)
- Map integration (show locations on map)
- Weather integration
- Photo galleries
- Export itineraries to PDF
- Mobile app version
